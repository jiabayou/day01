## 第一天 学习 Git
1. 现在学习git 命令
2. 复制/下载远程git仓库
	git clone [url]
3. 拉取远程仓库的代码
   	git pull [url]
4. 初始化git仓库吧
	git init 
5. 检查项目提交的状态
    git status
